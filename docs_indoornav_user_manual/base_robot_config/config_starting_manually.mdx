# Manually Starting IndoorNav {#config-starting-manually}

To manually start the IndoorNav software, you can run

```bash
roslaunch cpr_indoornav_ROBOT indoornav.launch
```

where `ROBOT` is your supported Clearpath platform:

-   `dingo`
-   `husky`
-   `jackal`
-   `ridgeback`
