IndoorNav is an all-in-one autonomy kit that enables robust point-to-point autonomous
navigation of mobile robots in indoor environments. It provides advanced software with
a powerful processor and sensor suite, and integrates seamlessly with Clearpath mobile
robotic development platforms. Designed for researchers in academia and corporate R&D,
IndoorNav is ROS ready and fully extensible with a well documented developer API.
