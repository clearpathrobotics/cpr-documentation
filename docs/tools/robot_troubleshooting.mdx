Refer to the [Physical Tools page](#common_hand_tools) for a list of tools that you may need to access the electronics inside your robot.

## Physical Tools

- Multimeter
- RJ45 Network cable
- Any converters for your development computer, to connect an RJ45 Network cables
- Monitor with a HDMI port
- HDMI cable
- Keyboard with a teathered USB cable (a mouse is not required)
