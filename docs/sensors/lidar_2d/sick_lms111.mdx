|  3  | Screw, Cap, Round Head—M5×0.8 X 8, Stainless Steel  |                        023316                        |    4     |
|  4  | Screw, Cap, Round Head—M5×0.8 X 12, Stainless Steel |                        023318                        |    4     |

### Instructions

1.  Fasten the bracket (ID 1) to the robot using M5 screws (ID 3)
    There is white painted lettering on the robot's top plate.
    This lettering designates sensor locations on the robot.
    Take note of the location that you have mounted this sensor.
    You will need this sensor location during software installation, to make sure your robot's URDF is configured correctly.
2.  Mount the sensor (ID 2) to the bracket (ID 1) using four M5 screws (ID 4).
3.  Connect a M12-4-D to RJ45 network cable to the sensor and robot's computer or network switch.
4.  Connect a M12-5-A power cable to the sensor, and robot's User Power breakouts. Ensure that the cable pinout is correct, and can supply 12 V or 24 V to the sensor.
