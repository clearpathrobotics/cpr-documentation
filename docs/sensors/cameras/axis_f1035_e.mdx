| Pin | Description   | Wire colour |
| :-: | :------------ | :---------- |
|  1  | Signal 1, -ve | Brown       |
|  2  | Signal 1, +ve | White-Brown |

|  4  |               | No Wire     |

The off-the-shelf camera comes from Axis with either a 3 metre, or 10 metre cable.
