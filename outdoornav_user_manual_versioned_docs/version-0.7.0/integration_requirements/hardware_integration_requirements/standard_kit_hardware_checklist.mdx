The OutdoorNav Standard Hardware Kit from Clearpath Robotics contains:

- SwiftNav Piksi/Duro GNSS (x2)
- Microstrain GX5-25 AHRS
- Velodyne Puck (VLP-16) 3D LiDAR

To integrate this kit onto your robot, ensure that all of the requirements below are satisfied for proper operation of the OutdoorNav software.
