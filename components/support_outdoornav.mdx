If possible, include as much of the following information as possible in your OutdoorNav support request:

- OutdoorNav software version
- Docker container states

  - This can be obtained by running the terminal command `docker ps` on the
    computer with OutdoorNav installed

- Docker log file

  - You can dump the current logs to a file by running the following terminal
    command on the computer with OutdoorNav installed:
    `cd ~/cpr_outdoornav_launch && docker compose --profile=outdoornav logs > $(date +"%Y_%m_%d_%I_%M_%p").txt`
