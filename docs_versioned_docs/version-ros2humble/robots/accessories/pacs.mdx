## Overview

The Platform Attachment Configuration System (PACS&trade;) is a Clearpath Robotics standard for adding components to a robot.
Our robots have mounting plates with a 80 mm X 80 mm grid of M5Ã—0.8 threaded holes.
This allows us to add sensors and attachments using adapter plates.
The _Parts Lists_ in our [Sensors pages](./sensors) include these 80 mm X 80 mm brackets.
Some example brackets are also detailed in the [Common PACS&trade; Items section](#pacs_common_pacs_items_for_all_robots) below.
